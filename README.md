# language-io package

Io Language support for Atom.

-----

Adds syntax highlighting and snippets to Io files in Atom

-----

## Install from Atom Package

From Atom Package search for "io"

## Install manual

cd ~/.atom/packages

git clone https://github.com/dotfelixb/language-io.git
